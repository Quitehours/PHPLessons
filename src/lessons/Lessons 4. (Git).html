Предварительная настройка при первом запуске:
git config --global user.name "Your Name"
git config --global user.email "your_email"
//в ковычках Имя и Почту

Работа с локальным репозиторием:
git init //инициализация локального репозитория
git status //проверяет состояние репозитория
git add [название файла/"." все файлы] //добавляет в промежуточную память
git commit { :w (записать), :q (выйти), :qa (выход без сохранения) }
git commit -m "краткое описание изменений" //коммит с сообщением
git commit -a -m "сообщение" //коммит вместе с добавлением в промежуточную память (не рекомендуемо к использованию)
git log // просмотр всех коммитов
git branch // показывает список всех веток в репозиторие и текущую ветку
git branch [имя] // создает новую ветку с именем [имя]
git checkout [имя] //переход на ветку с именем [имя]
git checkout -b [имя] //создает и переходит новую ветку с именем [имя]
git merge [имя] //"мерджит" ветку master с веткой [имя](необходимо быть в ветке master)
git branch -d [имя] //удаляет ветку [имя]

Работа с удаленным репозиторием:
git remote add origin [ссылка] //сохраняет ссылку на удаленный репозиторий с названием "origin"(можно изменить)
git push -u origin HEAD // "пушит"(отправляет) сохраненные коммиты на удаленный репозиторий и сохраняет его по умолчянию в дальнейшем
git push // "пушит"(отправляет) сохраненные коммиты на удаленный репозиторий используя ранние настройки
git pull //проверяет изменения на удаленном репозиторее. Если такие есть, загружает в локальный
git clone [ссылка] //копирует удаленный репозиторий на компьютер
